# Product Context: Quantum Swarm

## Description
Quantum Swarm is a hierarchical multi-agent financial analysis and trading system. It leverages the OpenClaw framework to orchestrate specialized AI agents that process market signals, manage risk, and execute trades autonomously.

## Primary Users
- **Quantitative Traders:** To automate complex trading strategies and factor analysis.
- **Financial Analysts:** To leverage AI for macro sentiment and trend analysis.
- **Developers:** To build and extend autonomous agentic frameworks for finance.

## Main Goal
The primary goal is to transition from static algorithmic trading to a dynamic, self-optimizing cognitive architecture that adapts to market regime changes while maintaining strict risk and regulatory guardrails.

## Main Features
- **L1 Strategic Orchestrator:** Intent classification, task decomposition, and capital allocation.
- **L2 Domain Managers:** Specialized agents for Macro Analysis, Quant Modeling, and Risk Management.
- **L3 Stateless Executors:** Dedicated workers for data fetching, backtesting, and order routing.
- **Self-Improvement Loop:** Automated logging of trades and indicator states to generate learned rules for future execution.
- **OpenClaw Gateway Integration:** Uses a standardized messaging and routing protocol for inter-agent communication.
- **Web Dashboard:** Real-time monitoring and control interface for the swarm.
