**Critical implementation details:**

- **Cosine distance** for sentiment: Use `collections.Counter` dot product. For two vectors `a` and `b` (dicts), compute `dot = sum(a[k]*b[k] for k in set(a)|set(b))`, then `mag_a = sqrt(sum(v*v for v in a.values()))`, `mag_b = sqrt(sum(v*v for v in b.values()))`. Distance = `1.0 - dot/(mag_a*mag_b)` if both nonzero, else 0.0.

- **Context-aware role boundary counting:** For each forbidden term found in THESIS_SUMMARY text, check a window of 5 tokens before and after. If any negation marker appears in that window, do NOT count as violation. If any assertion marker appears, count it. If neither assertion nor negation marker, do NOT count (require assertion context).

- **Proposal rejection rate:** Read all JSON files in proposals_dir. Filter by agent_id matching handle. Count `status=="rejected"` vs total. Use trailing_window config value for recency filter on `proposed_at`.

- **KAMI variance:** Read merit_scores from DB (not SwarmState — auditor is out-of-band). `SELECT dimensions FROM agent_merit_scores WHERE soul_handle=%s`. Parse JSONB dimensions dict, compute `statistics.variance([accuracy, recovery, consensus, fidelity])`.

- **Unsuspend CLI:** `--unsuspend MOMENTUM` sets `evolution_suspended=FALSE` in agent_merit_scores, resets all breach_counts to 0 in ars_state for that agent, appends `ARS_UNSUSPEND` audit event.

- **All file I/O is synchronous** (Path.read_text). DB operations use asyncio via get_pool(). The __main__ block uses asyncio.run().
