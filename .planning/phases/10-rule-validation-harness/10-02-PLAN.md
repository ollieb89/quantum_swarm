**Class RuleValidator:**

`__init__(self, config_path="config/swarm_config.yaml", registry_path="data/memory_registry.json", audit_path="data/audit.jsonl")`:
- Store self.audit_path = Path(audit_path)
- self.registry = MemoryRegistry(registry_path)
- Load YAML: cfg = yaml.safe_load(Path(config_path).read_text()) if Path(config_path).exists() else {}
- self.lookback_days = cfg.get("self_improvement", {}).get("validation_lookback_days", 90)
- self.min_trades = cfg.get("self_improvement", {}).get("validation_min_trades", 10)

`validate_proposed_rules(self) -> int` (synchronous, returns count of rules processed):
- Re-load registry to pick up recently persisted rules: self.registry.schema = self.registry._load()
- rules = self.registry.get_proposed_rules()
- For each rule: extract instrument from rule.condition.get("instrument", "AAPL")
- Run two backtests synchronously via asyncio.run() with asyncio.to_thread:
  - baseline = asyncio.run(asyncio.to_thread(_run_nautilus_backtest, instrument, {}))
  - treatment = asyncio.run(asyncio.to_thread(_run_nautilus_backtest, instrument, {"rule_id": rule.id, "rule_title": rule.title}))
  - Wrap both calls in try/except Exception: on error log and continue to next rule (leave proposed)
- Check baseline["total_trades"] < self.min_trades → log and skip
- Call self._passes_validation(baseline, treatment) to get bool
- On pass: self.registry.update_status(rule.id, "active")
- On fail: self.registry.update_status(rule.id, "rejected")
- Populate evidence: find updated rule via self.registry.get_rule(rule.id), set evidence dict with all six keys, call self.registry.save()
- Write audit event via self._write_audit(rule.id, outcome, baseline, treatment)
- Return count of rules processed

`_passes_validation(self, baseline: dict, treatment: dict) -> bool`:
- improvements = 0
- if treatment["sharpe_ratio"] > baseline["sharpe_ratio"]: improvements += 1
- if treatment["max_drawdown"] > baseline["max_drawdown"]: improvements += 1  # less negative = better
- if treatment["win_rate"] > baseline["win_rate"]: improvements += 1
- return improvements >= 2

`_write_audit(self, rule_id: str, outcome: str, baseline: dict, treatment: dict)`:
- Build event dict with all required fields (see RESEARCH.md Pattern 3)
- self.audit_path.parent.mkdir(parents=True, exist_ok=True)
- Append JSON line to self.audit_path
