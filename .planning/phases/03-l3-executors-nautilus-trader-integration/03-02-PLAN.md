    **`def _run_nautilus_backtest(symbol: str, strategy: dict) -> dict`** — the synchronous NautilusTrader worker (called via asyncio.to_thread):
    1. Fetch yfinance data: `df = yf.download(symbol, period="6mo", interval="1d", auto_adjust=True)`
    2. Lowercase columns: `df.columns = [c.lower() for c in df.columns]`
    3. Build `Equity` instrument using the pattern in RESEARCH.md `_build_equity_instrument(symbol, venue_name="SIM")`
    4. Build `BarType` and process with `BarDataWrangler`: `bars = wrangler.process(df)`
    5. Create `BacktestEngine` with `BacktestEngineConfig(bypass_logging=True)`. Add venue: `engine.add_venue(venue_name="SIM", oms_type=OmsType.NETTING, account_type=AccountType.CASH, starting_balances=[Money(100_000, USD)])`. Add instrument: `engine.add_instrument(instrument)`. Add data: `engine.add_data(bars)`.
    6. Run: `engine.run()` — this is synchronous and blocking; it is safe here because we are already in `asyncio.to_thread`.
    7. Extract metrics via `_extract_backtest_metrics(engine)` — returns a plain dict only. Do NOT return any NautilusTrader objects.
    8. Call `engine.dispose()` to release resources.

    **`def _extract_backtest_metrics(engine: BacktestEngine) -> dict`**:
