    The node:
    1. Reads `symbol` and `asset_class` from `state["quant_proposal"]` (default symbol="AAPL", asset_class="equity")
    2. Determines if fundamentals are requested from `state.get("metadata", {}).get("fetch_fundamentals", False)`
    3. Dispatches to the appropriate data source:
       - equity: `await fetch_equity_data(symbol)`
       - crypto: `await fetch_crypto_ohlcv(symbol)`
    4. Always fetches: `await fetch_news_sentiment(symbol)`, `await fetch_economic_data()`
    5. If fundamentals requested: `await invoke_dexter_safe(f"Fundamental analysis of {symbol}", symbol)`
    6. Returns:
       ```python
       {
           "data_fetcher_result": {
               "market": market_data.model_dump(),
               "sentiment": sentiment.model_dump(),
               "economic": economic.model_dump(),
               "fundamentals": fundamentals.model_dump() if fundamentals else None,
           },
           "messages": [{"role": "assistant", "content": f"DataFetcher: {symbol} ({asset_class}) data fetched"}]
       }
       ```
       IMPORTANT: Use `.model_dump()` (Pydantic v2), not `.dict()`. All returned values must be JSON-serializable plain dicts.
