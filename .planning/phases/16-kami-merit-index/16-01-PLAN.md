       Reads state.get("execution_result") or {}. Returns 1.0 if result["success"] is True.
       Returns 0.0 if error_type in {"INVALID_INPUT", "SCHEMA_FAILURE", "MALFORMED_OUTPUT", "TOOL_ERROR"}.

       Returns 0.0 for any other failure.

    6. _extract_consensus_signal(state: dict) -> float
       Reads state.get("weighted_consensus_score"). Returns 0.0 if None.
       Returns min(1.0, abs(score - 0.5) * 2.0) — normalized distance from neutral.
       This rewards convergence contribution without penalising contrarians (CASSANDRA protection).

       Uses Option A from RESEARCH.md: call load_soul(agent_id), return 1.0 if soul.identity.strip() else 0.0.
