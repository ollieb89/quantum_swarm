```python
def _calculate_hash(self, entry: Dict[str, Any], prev_hash: Optional[str]) -> str:
    data_string = json.dumps({...}, sort_keys=True)
    hasher = hashlib.sha256()
    hasher.update(data_string.encode('utf-8'))
    if prev_hash:
        hasher.update(prev_hash.encode('utf-8'))
    return hasher.hexdigest()
```

    Hashing flow (strictly ordered):
    1. Populate ALL card fields including prev_audit_hash
    2. card_dict = card.model_dump(mode="json")  # datetimes serialized to ISO strings
    3. payload = {k: v for k, v in card_dict.items() if k != "content_hash"}
    4. raw = canonical_json(payload)  # json.dumps(sort_keys=True, ensure_ascii=False, default=str)
    5. content_hash = hashlib.sha256(raw.encode("utf-8")).hexdigest()
    6. card.content_hash = content_hash

    Pydantic models (Pydantic v2 style, already in project):
    - AgentContributions(BaseModel): macro_report, quant_proposal, bullish_thesis, bearish_thesis, debate_resolution — all Optional[dict] = None
    - RiskSnapshot(BaseModel): consensus_score: float, risk_approval: dict, compliance_flags: list[str], portfolio_risk_score: Optional[float] = None
    - DecisionCard(BaseModel): card_id: str (UUID4 default), schema_version: str = "1.0", event_type: str = "decision_card_created", task_id: str, generated_at: datetime (UTC default), execution_result: dict, agent_contributions: AgentContributions, applied_rule_ids: list[str], risk_snapshot: RiskSnapshot, prev_audit_hash: Optional[str] = None, hash_algorithm: str = "sha256", content_hash: str = ""

    Module-level functions:
    - canonical_json(payload: dict) -> str: json.dumps(payload, sort_keys=True, ensure_ascii=False, default=str)
    - _compute_hash(card_dict: dict) -> str: exclude "content_hash" key, run canonical_json, SHA-256, return hexdigest
    - verify_decision_card(card_dict: dict) -> bool: recompute hash via _compute_hash, compare to card_dict["content_hash"], never mutate
    - build_decision_card(state: dict, registry=None, prev_audit_hash: Optional[str] = None) -> DecisionCard:

        - Construct DecisionCard with prev_audit_hash parameter
        - Compute content_hash: card_dict = card.model_dump(mode="json"); payload = {k: v for k, v in card_dict.items() if k != "content_hash"}; card.content_hash = hashlib.sha256(canonical_json(payload).encode("utf-8")).hexdigest()
