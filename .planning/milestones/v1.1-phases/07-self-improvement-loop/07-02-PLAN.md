The orchestrator reads data/MEMORY.md as raw text and injects it into agent prompts.
Reference text format (from crypto_learning.py MemoryManager.update_memory):
  - PREFER: {description}
  - AVOID: {description}
  - CAUTION: {description}

Map MemoryRule.type to prefix:
  "strategy_preference" with positive action bias  -> "PREFER: {title} — {condition} => {action}"
  "risk_adjustment" with avoid/reduce action       -> "AVOID: {title} — {condition} => {action}"
  "market_regime"                                  -> "CAUTION: {title} — {condition} => {action}"
  default (or "general")                           -> "CAUTION: {title} — {condition} => {action}"

Simpler approach: derive prefix from rule.action dict. If action contains "bias": "avoid*"
or "reduce*" -> AVOID. If action contains "bias": "long"|"buy"|"prefer" -> PREFER. Else CAUTION.
This keeps the format stable without adding a new field to MemoryRule.

{PREFIX}: {title}

4. Add a private helper `_rule_to_prefix(self, rule: MemoryRule) -> str` that maps a rule to PREFER/AVOID/CAUTION:
   - If `rule.type == "strategy_preference"` and action does NOT contain keys/values suggesting avoidance → return `"PREFER"`
   - If action dict values contain strings starting with "avoid" or "reduce" (case-insensitive) → return `"AVOID"`
   - If `rule.type == "risk_adjustment"` → return `"AVOID"` (risk adjustments are typically avoid-type)
   - Otherwise → return `"CAUTION"`

   Simplest correct implementation:
   ```python
   def _rule_to_prefix(self, rule: MemoryRule) -> str:
       action_vals = " ".join(str(v) for v in rule.action.values()).lower()
       if any(kw in action_vals for kw in ("avoid", "reduce", "short_only")):
           return "AVOID"
       if rule.type == "strategy_preference":
           return "PREFER"
       if rule.type == "risk_adjustment":
           return "AVOID"
       return "CAUTION"
   ```

5. Update `persist_rules()` to append to MEMORY.md after JSON registry save:
   ```python
   def persist_rules(self, rules: List[MemoryRule]):
       """Append new rules to the JSON registry and to data/MEMORY.md."""
       if not rules:
           return
       for rule in rules:
           self.registry.add_rule(rule)
       # Append human-readable entries to MEMORY.md for orchestrator injection
       self.memory_md_path.parent.mkdir(parents=True, exist_ok=True)
       timestamp = datetime.now(timezone.utc).strftime("%Y-%m-%dT%H:%M:%SZ")
       lines = [f"\n<!-- pipeline:{timestamp} -->"]
       for rule in rules:
           prefix = self._rule_to_prefix(rule)
           lines.append(f"- {prefix}: {rule.title}")
       with open(self.memory_md_path, "a") as f:
           f.write("\n".join(lines) + "\n")
       logger.info("Appended %d rules to %s", len(rules), self.memory_md_path)
   ```
